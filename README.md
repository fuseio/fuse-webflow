# Fuse Webflow

Custom code of Fuse Webflow website.

## Getting Started

Run development server:
```bash
npm run dev
```

## Contribute

Lint and format the code before committing.
1. Lint:
  ```bash
  npm run lint
  ```
2. Format:
  ```bash
  npm run format
  ```

## Deploy

Build the project:
```bash
npm run build
```
